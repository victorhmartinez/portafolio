Set de TV

El propósito de este proyecto es que los estudiantes de la Universidad Técnica Particular de Loja (UTPL) que están inscritos en la modalidad de educación abierta y a distancia puedan comprender las oportunidades que ofrece un set de televisión a través de un entorno virtual en 3D. En este espacio, los estudiantes pueden llevar a cabo diversas acciones, como ajustar la iluminación, manejar cámaras y trípodes de iluminación, así como explorar las herramientas y espacios reales, como camerinos, salas de control y los distintos escenarios utilizados en producciones televisivas.

Set TV Gamificado

Este proyecto es una extensión del proyecto original "SET TV" y ha sido enriquecido con nuevas características para convertirse en una experiencia gamificada. Entre las nuevas funcionalidades, destaca un sistema de diálogos en el que un avatar, que representa a un docente, guía a los usuarios proporcionando instrucciones para superar una serie de desafíos. Cada vez que los usuarios superan un desafío, obtienen recompensas. Además, se ha implementado un sistema de preguntas que utiliza información de una API externa alojada en los servicios de Firebase. Los usuarios deben elegir las preguntas adecuadas para llevar a cabo entrevistas de manera efectiva. Otra funcionalidad importante es el almacenamiento de datos que registra las acciones realizadas por los usuarios.

Set TV Multiusuario

El Set de Multiusuario es una adaptación de la versión original del Set TV que incorpora un sistema multiusuario. Ahora, permite que varios usuarios se conecten simultáneamente y interactúen entre sí mediante la tecnología de Photon (PUN2). Entre las funcionalidades de multiusuario se incluye la comunicación a través de chat de voz y texto. Esto se ha diseñado con el propósito de simular de manera más cercana una experiencia práctica en la vida real.

ARCSA-GAME

"ARCSA-Game" es un proyecto diseñado con el propósito de aumentar la conciencia de las personas acerca de los riesgos asociados a la compra de medicamentos en lugares no autorizados. Este proyecto se basa en el concepto de gamificación, integrando un sistema de retos y recompensas para motivar a los usuarios a tomar decisiones más informadas y seguras al adquirir medicamentos. Además, se ha implementado la funcionalidad de registro de usuarios con el fin de establecer un sistema de recompensas. Toda la información de los usuarios se almacena en una base de datos no relacional a través de Firebase.

Simulador de civil (mano de obra calificada)

El "Simulador de Mano de Obra Calificada" es un proyecto diseñado con la finalidad de ayudar a los estudiantes a comprender las diversas etapas que deben completarse antes de iniciar un proyecto de construcción. Este proyecto integra elementos de gamificación para hacer que el proceso de aprendizaje sea más interactivo y atractivo para los estudiantes. En este entorno, los estudiantes reciben retroalimentación por cada una de las actividades o misiones que deben llevar a cabo, lo que les permite aprender de manera efectiva y mejorar sus habilidades en el campo de la construcción.
